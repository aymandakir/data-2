# Crypto & Stock Price Analyzer

A professional Streamlit web application for analyzing cryptocurrency and stock price data. Features interactive dashboards, custom comparisons, and automated insights.

## Features

- **Dashboard**: Real-time market overview with KPI cards and interactive charts
- **Custom Compare**: Compare multiple tickers with normalized price charts
- **Auto Insights**: Automated analysis including best/worst performers, volatility, and MA crossover signals

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit app:
```bash
streamlit run app.py
```

## Usage

- **Dashboard**: View default 4 tickers (BTC-USD, ETH-USD, AAPL, TSLA) with price charts and metrics
- **Custom Compare**: Select any tickers from the sidebar and adjust the timeframe
- **Auto Insights**: See automated analysis of the default tickers
